# GPT Image 2 プロンプト集（日本語版）

[GPT Image 2](https://openai.com/) の日本語プロンプトを厳選キュレーションした、無料で使える日本語ギャラリーサイト。

🌐 **ライブサイト:** https://pokare.github.io/gpt-image-2-jp/

## このサイトについて

OpenAI の画像生成モデル「GPT Image 2」は、日本語テキストを画像内に正確にレンダリングできることで知られています。本サイトは、その能力を最大限に引き出すためのプロンプトカタログです。

- 📝 **126 件の厳選プロンプト**（元データ 2474 件から日本語優先でキュレーション）
- 🖼️ **157 枚のサンプル画像**
- 🇯🇵 **すべてのプロンプトが日本語**（コピーしてあなたの ChatGPT に貼るだけ）
- 🔍 **検索とカテゴリフィルター**で目的のプロンプトをすぐ見つけられる
- 💰 **完全無料、広告なし、登録不要**

## データソース

## ローカル開発

```bash
npm install
npm run update    # 元リポから最新データを取得・パース
npm run dev       # 開発サーバー起動
npm run build     # 本番ビルド
npm run preview   # ビルド成果物のプレビュー
```
